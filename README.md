MRCF
=========

A Collaborative Filtering Package based on Map/Reduce.

It contains the following implementations with different loss functions and optimization methods: 

Models:
###
		1. BasicMF
		2. FeatureMF
		3. SocialMF
		4. SVD++
		5. KNN

Loss:
###
		1. Square Loss
		2. Poisson Loss
		3. Pairwise Rank Loss
		
Optimization Methods:
###
		1. SGD
		2. Newton (Testing)
		3. ALS (Testing)
